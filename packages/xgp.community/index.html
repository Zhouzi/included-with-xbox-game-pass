Install the browser extension to add support for the Xbox Game Pass to Steam

This browser extension lets you know if the game you’re looking at is included with Xbox Game Pass so you don’t buy what you already own.

  • Add to Chrome
  • Add to Firefox

Review your wishlist

The browser extension also add a “included with Xbox Game Pass” badge in your wishlist. It’s an easy way to know if you’re waiting on games you could already be playing.

  • Add to Chrome
  • Add to Firefox

Don’t miss out on the new and upcoming games

Subscribe to the newsletter to receive a weekly summary of games that have been added to Xbox Game Pass as well as games being released soon.
